About Sky Vault

More than just file storage

Sky Vault is a full-stack cloud storage platform designed for both individual users and educational organisations. It combines secure file management with a complete organisation system for schools and institutions.

Whether you're a student storing assignments, a teacher sharing resources, or an admin managing your school's digital workspace — Sky Vault has you covered.

Features

Everything you need

A complete set of tools for personal file management and organisation-wide collaboration.

Secure Authentication

Firebase-powered login with email/password and Google OAuth. Strong password requirements enforced on all accounts.

File Upload & Storage

Upload any file type to Supabase Storage. Files are stored securely with unique paths per user — no cross-user access.

File Preview

Preview images, PDFs, videos, audio, and text files directly in the browser without downloading.

Folder Organisation

Create folders to organise your files. Navigate in and out of folders with a clean, intuitive interface.

Organisation System

Admins create organisations with unique join codes. Teachers and students join using the code and get role-based access.

Real-time Notifications

Class invites, assignment posts, and submissions appear instantly via Supabase Realtime — no refresh needed.

Class Management

Teachers create classes, post assignments, upload notes, and track student submissions with marks.

File Search

Search across all your files by name with live debounced results. Filter by file type — images, documents, videos, audio.

Trash & Restore

Deleted files go to trash first. Restore them anytime or permanently delete. Empty trash in one click.

How It Works

Up and running in minutes

Getting started is simple whether you're an individual user or setting up for your whole organisation.

1

Choose Your Role

Select whether you're an individual user or part of an organisation as a student, teacher, or admin.

2

Create Account

Sign up with email and password or continue with Google. Your account is secured by Firebase Authentication.

3

Join or Create Org

Admins create an organisation and get a unique join code. Teachers and students use that code to join.

4

Upload & Manage

Upload files, create folders, preview content, and collaborate with your organisation — all in one place.
